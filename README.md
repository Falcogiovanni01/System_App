# System_App
per compilare l'app: 
mvn clean package -DskipTests

per compilare i container:
docker-compose build
per avviare il container:
docker-compose up -d

https://localhost:8443/home

https://localhost:8443/admin/master/console/

https://localhost:9443/ui
